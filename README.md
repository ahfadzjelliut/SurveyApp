# SurveyApp
Project yang digunakan untuk pengunjung dalam mengisi survey <br />
Fitur Sebagai Pengunjung: <br />
-Memilih puas atau tidak puas <br />
-Memberikan komentar <br />

Fitur sebagai admin: <br />
-Melihat grafik lingkaran jumlah puas atau tidak puas <br />
-Melihat komentar
